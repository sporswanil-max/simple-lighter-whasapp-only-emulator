# simple-lighter-whasapp-only-emulator
specifically made only runnable whatsapp, if your like heck  i want whatsapp on my pc lightweight this might be the best choice, its only like 1 mb source and electron files are like 200mb.

portable running config


  npm install electron --save-dev
  npm start


creating app config


  npx @electron-forge/cli import
  npm run make
